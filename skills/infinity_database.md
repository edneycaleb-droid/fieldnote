# Infinity Database for AI-Native Applications

## Description
The Infinity database is a cutting-edge AI-native database designed to provide fast and efficient data retrieval and management for various LLM applications. It supports hybrid search capabilities for dense vectors, sparse vectors, tensors, and full-text data, making it an ideal solution for robust and efficient data management.

## Key Features
* Incredibly fast query latency and high throughput
* Supports hybrid search of dense embedding, sparse embedding, tensor, and full-text data
* Intuitive Python API for easy deployment and management

## Steps
### Deploy Infinity Database using Docker
1. Install Docker and pull the Infinity image
2. Run the Infinity container using the `docker run` command
### Install Infinity Client using Pip
1. Install the Infinity SDK using `pip install infinity-sdk`
### Create a Table and Insert Data
1. Connect to the Infinity database using the Python API
2. Create a table and insert data into the Infinity database
### Run a Vector Search Query
1. Use the Infinity API to run a vector search query

## Tools
* Docker: for deploying the Infinity database
* Infinity SDK: for interacting with the Infinity database using Python
* Python API: for creating and managing data in the Infinity database

## Concepts
* AI-native database: a database designed specifically for AI applications
* Hybrid search: a search capability that combines multiple data types
* Dense vector search: a search capability for dense vector data
* Sparse vector search: a search capability for sparse vector data
* Tensor search: a search capability for tensor data
* Full-text search: a search capability for full-text data
* LLM applications: applications that utilize large language models
