# Building Effective AI Agents for Automation

## Introduction
Building AI agents can help automate tasks, make decisions, and improve over time. In this skill, we will cover the fundamentals of building AI agents, including defining outcomes, creating identities, and equipping them with the right context and tools.

## Steps to Build an AI Agent
1. **Define the outcome and aim for a specific goal**: Clearly define what you want the agent to achieve. Use the rule of R to determine if a task is repetitive, rules-based, and generates a return on time.
2. **Create a definition of done (DOD)**: Define what success looks like for the agent. Make sure it is specific, measurable, and achievable.
3. **Use reverse prompting to clarify the outcome and requirements**: Ask the agent to ask you questions to clarify the outcome and requirements.
4. **Give the agent an identity**: Create a sole file, identity file, and user file to define the agent's personality, role, and context.
5. **Equip the agent with the right context, tools, and playbooks**: Provide the agent with the necessary context, tools, and playbooks to perform its tasks.
6. **Narrow the scope of the agent**: Focus the agent on a specific task or set of tasks to avoid confusing it.
7. **Consider using a manager agent**: Use a manager agent to coordinate sub-agents and tasks.
8. **Utilize the in-app browser for task automation and workflow optimization**: Use the in-app browser to automate tasks, access external tools, and optimize workflows.
9. **Create skills to automate repetitive tasks and workflows**: Create skills to automate repetitive tasks and workflows, and use the in-app browser to access relevant information.
10. **Use annotation tools for efficient communication with the AI agent**: Use annotation tools to communicate with the AI agent and provide feedback.
11. **Set up a new project in Claude to organize chats, files, and instructions**: Set up a new project in Claude to organize chats, files, and instructions around one ongoing task or topic.
12. **Use connectors to connect Claude with third-party tools**: Use connectors to connect Claude with third-party tools and access information from those tools.
13. **Turn a workflow into a skill for repeatable tasks**: Turn a workflow into a skill for repeatable tasks, allowing you to execute the workflow with a single command.
14. **Schedule tasks to run automatically at a specified time**: Schedule tasks to run automatically at a specified time, allowing you to automate tasks and workflows.

## Tools and Concepts
* Anthropic: A platform for building AI agents
* Haiku, Sonnet, Opus, Fable: AI models for different tasks and complexity levels
* Zoom: A video conferencing tool for recording processes
* Gmail: An email platform for connecting with the agent
* Definition of done (DOD): A clear definition of success for the agent
* Identity files: Sole file, identity file, and user file for defining the agent's personality, role, and context
* Context window: The agent's workspace for performing tasks
* Playbooks: Procedures for the agent to follow
* Manager agent: An agent that coordinates sub-agents and tasks
* Connectors: Tools for connecting Claude with third-party tools
