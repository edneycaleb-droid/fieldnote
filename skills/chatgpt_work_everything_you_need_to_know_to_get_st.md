# Getting Started with ChatGPT Work: A Beginner's Guide

## Description
ChatGPT Work is a powerful tool that allows you to automate tasks, build websites, and create content without needing to code. In this skill, we will cover the basics of ChatGPT Work, including how to activate the work mode, select tools, and give tasks to the AI.

## Steps
1. **Sign up for a ChatGPT account and activate the work mode**: To get started with ChatGPT Work, you need to sign up for a ChatGPT account and activate the work mode. This can be done by toggling the switch at the top of the ChatGPT interface.
2. **Select the sites tool to build a website**: To build a website, you need to select the sites tool. This can be done by typing `/sites` in the ChatGPT interface.
3. **Type in a prompt to give the AI a task**: Once you have selected the sites tool, you can type in a prompt to give the AI a task. For example, you can type `Build a website for my business`.
4. **Use the slash command to access presets and tools**: The slash command can be used to access presets and tools in ChatGPT Work. For example, you can type `/sites` to access the sites tool.
5. **Review and customize the output to fit your needs**: Once the AI has generated the output, you can review and customize it to fit your needs.
