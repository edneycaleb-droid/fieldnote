# Building AI Agents with the Pydantic AI Framework

Pydantic AI teaches how to build type-safe, model-agnostic, and production-ready AI agents using the Pydantic AI Framework.

## Steps
- Define a Pydantic AI agent with a model and instructions
- Register static instructions using keyword arguments or dynamic instructions with dependency injection
- Integrate tools and dependencies using the `tool` decorator and `RunContext` parameter
- Validate model outputs using Pydantic and `output_type` parameter
