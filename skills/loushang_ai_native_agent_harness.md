# Building and Utilizing AI-Native Agent Harness for Coding Workflows

## Description
Loushang is a method-native AI work system for running complex work from intent to verified delivery, focusing on multi-model LLM orchestration, stateful sessions, tool governance, and traceable delivery. It helps developers build reliable, recoverable, auditable, and deliverable AI-driven projects. By utilizing Loushang, developers can create complex coding workflows with features like multi-model LLM orchestration, stateful sessions, and tool governance.

## Steps
1. **Clone the Loushang repository**: Use `git clone https://github.com/zhnt/loushang.git` to download the repository.
2. **Create a virtual environment**: Run `uv venv .venv` to create a virtual environment and activate it using `source .venv/bin/activate`.
3. **Install the project**: Install the project in editable development mode using `uv pip install -e '.[dev]'`.
4. **Install Loushang**: Install Loushang from source using git clone and uv pip install.
5. **Explore available commands**: Run `loushang --help` to explore available commands and options.
6. **Run Loushang**: Run Loushang using the CLI and terminal workbench.
7. **Test the agent's functionality**: Use `loushang -p 'Inspect this repository and summarize what it does.'` to test the agent's functionality.
8. **Utilize Sessions**: Utilize sessions for persistent coding conversations and execution records.
9. **Govern Tools and Extensions**: Govern tools and extensions for customizable workflows.
