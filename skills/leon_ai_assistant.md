# Leon AI Assistant

## Description
Leon is an open-source personal AI companion that can understand goals, decide how to handle them, use tools, remember useful information, and recover from failures.

## Steps
1. Clone the Leon AI Assistant repository
2. Install dependencies
3. Run Leon AI Assistant in controlled mode
4. Use Leon AI Assistant with tools to get work done
5. Recover from failures
6. Work with local models and context
7. Use agent mode for step-by-step planning
8. Use real tools with context-aware responses
9. Run Leon AI Assistant with Docker
10. Install native skills for controlled actions
11. Install agent skills for workflows
12. Check the setup using `pnpm run check`
