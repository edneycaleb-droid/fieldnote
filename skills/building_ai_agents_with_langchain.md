# Building AI Agents with LangChain

## Description
LangChain is a framework for building applications powered by Large Language Models (LLMs). It provides a standard interface for models, embeddings, and vector stores, making it easier to develop AI applications. This skill covers the basics of using LangChain to build AI agents with real-time data augmentation, model interoperability, and rapid prototyping capabilities.

## Steps
1. Install LangChain using pip: `uv add langchain`.
2. Initialize a chat model using LangChain: `from langchain.chat_models import init_chat_model`.
3. Invoke the chat model: `model.invoke('Hello, world!')`.
4. Explore Deep Agents for higher-level agent capabilities.
5. Use LangGraph for building controllable agent workflows.

## Tools
* LangChain: A framework for building LLM-powered applications.
* Deep Agents: A higher-level package for building agents with common usage patterns.
* LangGraph: A framework for building controllable agent workflows.
* LangSmith: A platform for developing, debugging, and deploying AI agents and LLM applications.

## Concepts
* Large Language Models (LLMs): AI models that can process and generate human-like language.
* Model Interoperability: The ability to swap models in and out of an application.
* Real-time Data Augmentation: The ability to connect LLMs to diverse data sources and external/internal systems.
* Rapid Prototyping: The ability to quickly build and iterate on LLM applications.
* Agent Orchestration: The ability to build controllable agent workflows.
