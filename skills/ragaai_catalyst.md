# RagaAI Catalyst

## Description

RagaAI Catalyst is a comprehensive platform designed to enhance the management and optimization of LLM projects. It offers a wide range of features, including project management, dataset management, evaluation management, trace management, prompt management, synthetic data generation, and guardrail management.

## Steps

### Installation

1. Install RagaAI Catalyst using pip.

### Configuration

1. Set up credentials by setting environment variables or passing them directly to the RagaAICatalyst class.

### Project Management

1. Create and manage projects using RagaAI Catalyst.

### Dataset Management

1. Manage datasets efficiently for your projects.

### Evaluation Management

1. Create and manage metric evaluation of your RAG application.
