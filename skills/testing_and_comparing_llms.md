# Bonzai 27B

## Description
This skill teaches how to deploy and test Bonzai 27B, a 27 billion parameter model, on a CPU only i7 with 16 gigs of RAM.

## Steps
1. Download the Llama.cpp zip, the ternary model file, and the vision model.
2. Extract the zip and start the server.
3. Run the model on a CPU only i7 with 16 gigs of RAM and test its performance.

* Llama.cpp
* Prism ML
* Hugging Face
