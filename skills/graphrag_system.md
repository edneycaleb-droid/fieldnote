# GraphRAG: A Modular Graph-Based Retrieval-Augmented Generation System

## Overview
GraphRAG is a data pipeline and transformation suite that is designed to extract meaningful, structured data from unstructured text using the power of LLMs.

## Quickstart
To get started with the GraphRAG system, we recommend trying the command line quickstart.

## Prompt Tuning
Using GraphRAG with your data out of the box may not yield the best possible results. We strongly recommend to fine-tune your prompts following the Prompt Tuning Guide in our documentation.
## Steps
1. To run the quickstart, navigate to the GraphRAG repository directory and use `graphrag init --root [path] --force`.
2. After initiating the configuration, run `graphrag quickstart` to start the GraphRAG system.

## Tools
* GraphRAG: for initiating the system and setting up the required infrastructure.
* Command Line Interface: for executing the GraphRAG commands.
