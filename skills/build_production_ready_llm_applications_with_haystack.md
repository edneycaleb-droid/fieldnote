# Building Production-Ready LLM Applications with Haystack

## Description
Haystack is an open-source AI orchestration framework for building production-ready LLM applications in Python. It allows designers to create modular pipelines and agent workflows with explicit control over retrieval, routing, memory, and generation. By using Haystack, you can design scalable and customizable AI systems that can be easily integrated with various large language models and tools.

## Steps
### Installation
1. Install Haystack via pip: `pip install haystack-ai`
2. Install nightly pre-releases: `pip install --pre haystack-ai`

### Designing Pipelines and Agent Workflows
1. Design modular pipelines and agent workflows with explicit control over retrieval, routing, memory, and generation
2. Use built-in components for retrieval, indexing, tool calling, memory, and evaluation, or create custom components
3. Add loops, branches, and conditional logic to precisely control how context moves through pipelines and agent workflows
4. Create custom components using Haystack's extensible ecosystem and model- and vendor-agnostic design
5. Integrate Haystack with various large language models and tools, such as OpenAI, Mistral, Anthropic, Cohere, Hugging Face, Azure OpenAI, and AWS Bedrock

### Tooling and Deployments
1. Use Docker to containerize and deploy Haystack-based applications
